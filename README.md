# NebulaLearn

An all-in-one learning platform with courses, competitions, live webinars, and more!

## Features

- 📚 **Structured Learning Paths**: Curated courses from beginner to pro
- 🏆 **Hackathons & Competitions**: Compete with prizes and leaderboards
- 📡 **Live Webinars**: Weekly sessions with industry mentors
- 💾 **Cloud Drive**: Store your project files securely
- 🚀 **Project Showcase**: Display your GitHub projects
- 🎯 **Skill Wallet**: Track your progress and achievements
- 🔄 **Real-time Sync**: Live feed and updates every second!

## Tech Stack

- **Framework**: React + TanStack Router
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Backend**: Supabase
- **Deployment**: Vercel

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit http://localhost:8080

### Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/     # UI and feature components
├── contexts/       # React contexts
├── hooks/          # Custom hooks
├── integrations/   # Supabase and other integrations
├── lib/            # Utilities and business logic
├── routes/         # Page routes
└── styles.css      # Global styles and theme
```
